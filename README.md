This is the official repo for Vou de ônibus web app version.

## How to make this project works properly

Please, follow the steps bellow. :)

### Requirements

To execute this project you'll need:

* Node.js 5+
* Yarn

You can clone this repo now, and...

```bash
npm i
```
Or...

```bash
yarn
```
